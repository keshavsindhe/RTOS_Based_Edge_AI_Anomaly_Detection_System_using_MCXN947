# Edge AI Vibration Project

## Overview
This is an embedded firmware project for AI-based vibration monitoring on the FRDM-MCX-N947 board.

## Project Structure
- `edge_ai_vibration_cm33_core0/` - Core 0 application code
- `edge_ai_vibration/` - Peripheral drivers and utilities
- `CMakeLists.txt` - Build configuration

## Build Instructions
Use MCUXpresso IDE or CMake to build the project.

## Setup
1. Install MCUXpresso IDE
2. Install MCX-N947 SDK
3. Configure your SDK path in `.vscode/settings.json`
